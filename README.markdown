## Requirements

* Ruby 1.9.3
* MySQL
* rbenv
* sphinxsearch
* nodejs
* pdf2htmlEX

## development setup
* sudo apt-get install sphinxsearch
* sudo apt-get install nodejs
* rake db:setup
* rake ts:configure
* rake ts:index
* rake ts:start

## install pdf2htmlEx
* sudo apt-get install software-properties-common
* sudo add-apt-repository ppa:coolwanglu/pdf2htmlex
* sudo apt-get update
* sudo apt-get install pdf2htmlEX
* pdf2htmlEX -v
